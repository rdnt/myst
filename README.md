Make sure you have Golang & Node.js w/ NPM installed.  
Then:
- Install fswatch:  
`go get -u -v github.com/codeskyblue/fswatch`
- Install vue-cli:  
`npm install -g @vue/cli`
npm install -g @vue/cli-service-global

cd vue
npm install
